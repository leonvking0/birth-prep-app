# 01

这份产前课程涵盖了从临产准备到分娩过程中的医疗干预等大量细节。以下是根据课堂记录整理的关键信息总结：

## 一、 何时前往医院 (5-1-1 法则与特殊情况)

- 
    
    **宫缩计时 (5-1-1)：** 当宫缩每 **5 分钟**一次、每次持续 **1 分钟**、且这种情况持续 **1 小时**时，应前往医院 。
    
- 
    
    **无法言语原则：** 如果宫缩强烈到让你**无法说话或呼吸**，无论计时如何，都应立即就医 。
    
- **破水处理：**
    - 
        
        **不足 37 周：** 属于早产迹象，必须立即去医院 。
        
    - 
        
        **足月 (37 周+)：** 如果没有宫缩，可以观察。但若伴有**羊水异味、颜色异常（可能含有胎便）**，或产妇 **B 族链球菌 (GBS) 检测呈阳性**，必须立即入院接受抗生素治疗 。
        
    - 
        
        **确认方法：** 若不确定是否破水，可在内裤里垫纸巾，站起来提腹并咳嗽；若有液体涌出，则很可能已破水 。
        
- 
    
    **非临产预警症状：** 持续性头痛、面部或肢体突然肿胀、视力出现斑点、脚部发痒、发烧、呕吐或尿灼热（尿路感染），均需就医检查 。

## 二、 产前准备清单

- **心理与环境准备：**
    - 完成所有“未竟之事”（如税务、工作交接、甚至剪发），以免这些琐事成为产程中的心理障碍 。
    - 
        
        **筑巢本能：** 至少整理出一间让你感到舒适、安全的房间 。
        
- **物品准备：**
    - 
        
        **汽车座椅：** 提前安装并练习使用（建议看 YouTube 教程），出院时婴儿不可穿着厚重的雪地服坐在座椅内，以免安全带滑落 。
        
    - 
        
        **医院包：** 准备护手霜、舒适的睡衣、带有吸管的水瓶 。
        
    - 
        
        **食物：** 准备常温零食和冷冻肉汤（牛、鸡或蔬菜汤，加入姜和蜂蜜），以便在医院冲泡 。
        
- 
    
    **产后规划：** 研究催乳食物 (Galactagogues)，预约新生儿医生，并确认保险是否涵盖吸乳器租赁 。

## 三、 入院后的操作与应对

- **分娩评估 (Triage)：**
    - 
        
        **不要一直坐着：** 鼓励走动、使用分娩球或进行背部按摩，这比静坐更有助于产程进展 。
        
    - 
        
        **胎心监测：** 基准监测通常需要 30 分钟。如果宝宝在睡觉导致心率变异性低，可以喝**苹果汁或嚼冰块**来唤醒宝宝 。
        
    - 
        
        **入院标准：** 通常宫口开至 **3-4 厘米** 且宫缩规律强劲时才会正式安排进入产房 。
        
- **心理应对技巧：**
    - 
        
        **“向上看”法 (Eyes Up)：** 思考问题时抬头看，能帮助大脑进入创意、放松的频率 。
        
    - 
        
        **便利贴法：** 在便利贴上写下产后想和宝宝一起做的趣事，分娩时贴在墙上，利用感官深呼吸法进行冥想 。

## 四、 医疗干预：无痛分娩与缩宫素

### 无痛分娩 (Epidural)

- 
    
    **最佳时机：** 当你感到**从“应对”转为“受苦”**时（通常是 3 厘米左右）。过早使用（如 0 厘米）可能导致剖宫产风险增加 **50%**，因为限制了产妇的活动能力 。
    
- **副作用与风险：**
    - 
        
        **体位限制：** 需每半小时左右翻身一次，防止因压迫神经导致产后暂时性腿部麻木 。
        
    - 
        
        **体液影响：** 需使用临时导尿管排空膀胱，以便胎儿下降 。
        
    - 
        
        **脊髓性头痛：** 极少数情况下会出现，需通过“血贴” (Blood Patch) 手术治疗 。
        
- 
    
    **特殊限制：** 背部有大面积纹身（尤其是含深色墨水且位置正对着穿刺点）或接受过脊柱融合手术的人，可能无法进行无痛分娩 。
    

### 缩宫素 (Pitocin)

- 
    
    **用途：** 如果宫缩不够强劲或产程停滞，医生会使用人工缩宫素来增强频率和强度 。
    
- 
    
    **监测：** 使用缩宫素时必须进行持续胎心监测，因为它可能给宝宝带来压力；如果宝宝不耐受，护士会暂时停药 。

## 五、 医院环境与人员

- 
    
    **人员构成：** 你可能会遇到住院医生 (Residents) 或医学生。你有权询问他们的身份，并可以要求学生离开房间，或者要求由经验更丰富的护士协助 。
    
- 
    
    **特别提醒：** 某些医院（如 Sunnybrook）**不提供**新生儿割礼服务，有文化需求者需提前联系外部诊所 。
